					

					目录文档说明
							——Create by Damon.dai 2014-12-30
main: 为入口
	index.html：
		导航目录主页（请使用该页进行导航）
	index-dev.html
		为开发版本（地址前面加 ?dev 则会跳转开发版本）
	

API.DOC：
	里面放各种组件API文档说明 word文档格式（做调研用）

app  ： 
	里面放静态页面，使用demo.html。（只放html页面）

module：
	放公用组件框架，例：jquery，seajs
	jq 地址	F:\FeiNiu-local\feiniu-module\module\jquery\1.10.1
	（该文件夹请不要修改）
	封装好的js 文件
	（注：  只放js文件）
		

static：
	按照项目放项目对应的程序员编写的 css和js文件，
	或者gruntfile，package，config等nodejs所需要的配置文件
	（注：  img css 等应用所需文件）




公用文件:
	module/jquery/1.11.2/jquery-1.11.2.js
